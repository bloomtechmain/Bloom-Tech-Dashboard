# BloomAudit Dashboard (TypeScript Edition)

A professional PERN stack dashboard rebuilt from the ground up with native TypeScript support.

## Project Structure

- `client/`: React + Vite + TypeScript + Tailwind CSS
- `server/`: Node.js + Express + TypeScript + PostgreSQL

## Setup

1. **Install Dependencies**:
   ```bash
   npm run install-all
   ```

2. **Database Configuration**:
   Update `server/.env` with your PostgreSQL credentials.

3. **Development Mode**:
   ```bash
   npm run dev
   ```

## Key Features

- **End-to-End Type Safety**: Shared types and consistent TS architecture.
- **Premium UI**: Glassmorphic design with Tailwind CSS.
- **Modern Tooling**: Powered by Vite and tsx for near-instant developer feedback.
